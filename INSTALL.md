# 安装指南

## 快速开始

### 1. 准备文件

确保项目目录包含以下**必需文件**：
- ✅ `manifest.json` - 扩展配置文件
- ✅ `content.js` - 核心功能脚本
- ✅ `styles.css` - 样式文件

**可选文件**（不影响功能）：
- `icon16.png` - 16x16 图标
- `icon48.png` - 48x48 图标
- `icon128.png` - 128x128 图标

> 💡 **提示**：如果没有图标文件，扩展仍可正常工作，Chrome 会使用默认图标。如需添加图标，请参考 `ICONS.md` 文件。

### 2. 加载扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角开启**开发者模式**（Developer mode）开关
4. 点击**加载已解压的扩展程序**（Load unpacked）
5. 选择本项目的根目录（包含 `manifest.json` 的文件夹）
6. 扩展安装完成！

### 3. 验证安装

1. 访问 [Google Trends](https://trends.google.com)
2. 搜索任意关键词或对比多个关键词
3. 查看页面右上角是否出现悬浮窗
4. 鼠标在图表上移动，观察数值是否实时更新

## 故障排除

### 问题：扩展没有显示

**可能原因：**
- 页面未完全加载
- 扩展未正确安装
- 页面 URL 不匹配

**解决方法：**
1. 刷新页面（F5 或 Cmd+R）
2. 检查扩展是否在 `chrome://extensions/` 中已启用
3. 确认访问的是 `https://trends.google.com` 或 `https://trends.google.com.cn`
4. 打开浏览器开发者工具（F12），查看 Console 是否有错误信息

### 问题：悬浮窗显示"等待数据加载"

**可能原因：**
- 页面结构发生变化
- 数据还未加载完成
- DOM 选择器无法匹配当前页面结构

**解决方法：**
1. 等待几秒钟，让页面完全加载
2. 尝试在图表上移动鼠标，触发工具提示
3. 如果持续无法读取数据，可能需要更新选择器逻辑

### 问题：数值不准确

**说明：**
- 显示的搜索量是**估算值**，基于线性映射假设
- 不同词条之间的换算可能存在偏差
- 时间范围或地域变化会影响准确性

**建议：**
- 使用已知搜索量的词条作为校准点
- 在同一时间范围和地域设置下使用
- 理解这是估算值，不是官方数据

### 问题：悬浮窗位置不合适

**解决方法：**
- 点击悬浮窗标题区域并拖拽到合适位置
- 刷新页面后位置会重置到默认位置（右上角）

## 卸载扩展

1. 访问 `chrome://extensions/`
2. 找到 "Trends Index → Volume Overlay" 扩展
3. 点击**移除**（Remove）按钮
4. 确认删除

## 更新扩展

1. 在 `chrome://extensions/` 页面找到扩展
2. 点击**重新加载**（Reload）按钮
3. 或重新执行"加载已解压的扩展程序"步骤

## 开发调试

### 查看扩展日志

1. 在 `chrome://extensions/` 页面，找到扩展并点击**检查视图：content script**
2. 或打开页面后按 F12，在 Console 中查看日志

### 修改代码后重新加载

1. 修改代码文件（`content.js`、`styles.css` 等）
2. 在 `chrome://extensions/` 页面点击扩展的**重新加载**按钮
3. 刷新 Google Trends 页面

## 技术支持

如果遇到问题：
1. 检查浏览器控制台是否有错误信息
2. 确认扩展版本和 Chrome 版本兼容
3. 查看 README.md 了解详细功能说明
