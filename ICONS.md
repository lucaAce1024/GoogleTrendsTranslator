# 图标文件说明

## 图标要求

Chrome 扩展需要以下尺寸的图标文件：

- `icon16.png` - 16x16 像素（扩展列表中的小图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome Web Store）

## 创建图标

你可以使用以下方法创建图标：

### 方法一：在线工具
1. 访问 [Favicon Generator](https://realfavicongenerator.net/) 或类似工具
2. 上传一个 128x128 或更大的图片
3. 生成所需尺寸的图标

### 方法二：设计工具
使用 Figma、Photoshop、GIMP 等工具创建：
- 设计一个简单的趋势图表图标
- 导出为 PNG 格式
- 调整到所需尺寸

### 方法三：临时占位
如果暂时没有图标，可以：
1. 创建简单的纯色 PNG 图片
2. 或者从 `manifest.json` 中暂时移除 `icons` 字段（扩展仍可正常工作）

## 图标建议

- 使用与 Google Trends 相关的视觉元素（如折线图、趋势箭头）
- 保持简洁，在小尺寸下也能清晰识别
- 使用与扩展功能相关的颜色（如蓝色、绿色）
